## DICAS

- **Midjourney:** adicione `--ar 16:9` para backgrounds, `--ar 4:3` para carros, `--ar 1:1` para circuitos
- **DALL-E 3:** prompts em inglês funcionam melhor, seja específico na cor do neon
- **Adobe Firefly:** ative "No content credentials" e "Photorealistic" style
- **Leonardo.ai:** use modelo "PhotoReal" para carros, "DreamShaper" para circuitos
- **Resolução mínima:** 1920×1080 para backgrounds, 1200×900 para carros e circuitos
- Após gerar, solte o arquivo em `public/images/` com o nome exato — o código já está mapeado!
